Teoria JavaScript

1. Kiedy powstał JavaScript?

JavaScript został stworzony w 1995 roku.

2. Dlaczego jest to język skryptowy?

JavaScript jest językiem skryptowym, ponieważ jest interpretowany i nie wymaga kompilacji.

3. Gdzie jest wykonywany JS (klient lub serwer)?

JavaScript jest wykonywany po stronie klienta, w przeglądarce użytkownika.

4. Dlaczego JS to bezpieczne narzędzie?

JavaScript jest bezpieczny, ponieważ ma ograniczony dostęp do zasobów lokalnych komputera.

5. Czy jest to język obiektowy, jeśli tak, to dlaczego?

Tak, JavaScript jest językiem obiektowym, ponieważ wykorzystuje definicje obiektów, metod i zdarzeń.
